*****{Обработка ошибок}***Появился метод жизненного цикла {componentDidCatch()} для отлавливания ошибок. Также дерево DOM отмонтируется и выводится резервный интерфейс. Есть возможность отследить ошибки через границы ошибок {error boundaries} - это компоненты React, которые отлавливают ошибки JavaScript в любом месте их дочернего дерева компонентов, регистрируют эти ошибки и отображают резервный интерфейс вместо поломанного дерева компонентов, т.е. если какой-то компонент в одной из этих областей пользовательского интерфейса ломается, остальные продолжают исправно работать.

*****{Порталы} позволяют присоединить данные к внешнему элементу в дереве DOM
